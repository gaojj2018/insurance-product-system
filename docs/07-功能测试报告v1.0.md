### 修复详情

#### ✅ 已修复 (代码层面)

1. **财务报表axios错误**
   - 文件: `frontend/src/views/FinanceList.vue`
   - 修改: 添加 `getFinanceStats` 导入，修改 `loadStats` 函数使用正确的API
   - 状态: ⚠️ 代码已修复，需清除浏览器缓存或重启nginx后生效

2. **Gateway路由配置**
   - 文件: `gateway/src/main/resources/application.yml`
   - 修改: 添加 `/api/finance/stats` 和 `/api/finance/report` 路由
   - 状态: ✅ 已修复，需重启gateway服务

#### ❌ 未修复

1. **机构管理/角色管理 403错误**
   - Spring Security拦截原因: 后端了未认证请求
   - 解决: 需要确保登录成功后正确传递JWT Token

2. **理赔款管理 404错误**
   - 原因: `finance-service` 缺少 `ClaimPaymentController`
   - 解决: 需要在后端创建理赔支付控制器

3. **核保管理数据缺失**
   - 原因: `Underwriting` 实体只存储了applicationId，需要关联查询
   - 解决: 需要在后端修改Service层添加关联查询
