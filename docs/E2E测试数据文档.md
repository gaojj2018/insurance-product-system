## 一、数据结构说明

### 1. 产品管理 (insurance_product)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 产品ID |
| product_code | VARCHAR | 产品代码 |
| product_name | VARCHAR | 产品名称 |
| product_type | VARCHAR | 产品类型 |
| coverage_period | VARCHAR | 保障期间 |
| payment_period | VARCHAR | 缴费期间 |
| min_coverage | DECIMAL | 最低保额 |
| status | VARCHAR | 状态(在售/停售) |

### 2. 投保申请 (insurance_application)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 申请ID |
| application_no | VARCHAR | 申请单号 |
| product_id | BIGINT | 产品ID |
| applicant_name | VARCHAR | 申请人姓名 |
| id_card | VARCHAR | 身份证号 |
| phone | VARCHAR | 手机号 |
| coverage | DECIMAL | 保额 |
| premium | DECIMAL | 保费 |
| status | VARCHAR | 状态 |

### 3. 核保管理 (insurance_underwriting)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 核保ID |
| application_id | BIGINT | 投保申请ID |
| uw_result | VARCHAR | 核保结果 |
| uw_remark | VARCHAR | 核保备注 |

### 4. 保单管理 (insurance_policy)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 保单ID |
| policy_no | VARCHAR | 保单号 |
| application_id | BIGINT | 投保申请ID |
| applicant_name | VARCHAR | 投保人 |
| insured_name | VARCHAR | 被保人 |
| coverage | DECIMAL | 保额 |
| premium | DECIMAL | 保费 |
| status | VARCHAR | 状态 |

### 5. 理赔管理 (insurance_claims)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 理赔ID |
| claim_no | VARCHAR | 理赔单号 |
| policy_id | BIGINT | 保单ID |
| claim_amount | DECIMAL | 理赔金额 |
| claim_reason | VARCHAR | 理赔原因 |
| status | VARCHAR | 状态 |

### 6. 客户管理 (insurance_customer)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 客户ID |
| customer_name | VARCHAR | 客户姓名 |
| id_card | VARCHAR | 身份证号 |
| phone | VARCHAR | 手机号 |
| email | VARCHAR | 邮箱 |
| address | VARCHAR | 地址 |
| status | VARCHAR | 状态 |

### 7. 财务管理 (insurance_finance)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 记录ID |
| policy_no | VARCHAR | 保单号 |
| premium | DECIMAL | 保费 |
| payment_status | VARCHAR | 支付状态 |
| payment_date | DATE | 支付日期 |
